HL7 Segment

HL7 messages is composed of several lines of data, each separated by a carriage return. Each line of data is called a Segment. Segments begin with a 3 character segment name and then a set of data fields. The name and format of the segment is defined by HL7 standards.

Resolution

The DQA follows HL7 and National standards when reading HL7 segments. Please consult the appropriate standards documents to verify that the message is using the correct segments, in the correct order.
