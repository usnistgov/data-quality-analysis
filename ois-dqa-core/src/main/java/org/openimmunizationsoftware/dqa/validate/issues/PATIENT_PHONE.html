Patient Phone

The primary phone for the patient or the patient's parent/guardian.

The DQA validates phone numbers for those using the North American Numbering Plan. All other phone numbers, such as international phone numbers, are not currently validated. The following basic checks are made if the phone number is not empty:

  • If area code is specified then must be 3 digits long and the first digit may not be a 1 or 0.
  • The local number must contain exactly 7 numeric digits. Other non-digit characters are ignored.
  • The 1st local number digit can not be a 1 or a 0.
  • The 2nd and 3rd local number can not be both 1. (For example: 411 and 511 are not valid exchange numbers.)
