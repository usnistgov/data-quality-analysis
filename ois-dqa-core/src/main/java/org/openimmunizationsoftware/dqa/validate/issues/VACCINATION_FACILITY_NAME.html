Vaccination Facility Name

The name of the facility where the vaccination was administered. This should only be used to indicate a facility within or indicating the submitting organization. For historical vaccinations, those give outside of the submitting organization jurisdiction, the name of the facility should be left blank. Any other value will normally be ignored.

Important Note: Issues associated with this field are only triggered if the vaccination is indicates has having been administered. Historical vaccinations are not expected to be reported with this information.
